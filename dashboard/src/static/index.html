How it works

Everything your leak community needs

A complete monitoring system — from RSS polling to Discord delivery — with one dashboard to manage it all.

⚡
Real-Time Polling
Polls leaked.cx RSS feeds every 35 seconds. Premium members get priority 15-second intervals so they're always first.
🎤
Artist Tracking
Add artists by name. We normalize and match automatically — no exact spelling required. Assign a Discord role for instant pings.
🔔
Rich Notifications
Beautifully formatted Discord embeds with title, tags, matched artists, and direct links. No noise, just signal.
🗑️
Deletion Tracking
When a thread disappears from leaked.cx, Prefix Hub flags it instantly — so you know what got taken down.
🔍
Advanced Filters
Whitelist or blacklist artists, filter by tag, set minimum age thresholds. Premium-exclusive for maximum control.
📊
Analytics Dashboard
Track notification volume, feed health, and artist activity over time — all from one clean dashboard.
